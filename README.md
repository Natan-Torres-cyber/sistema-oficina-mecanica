# Sistema de Gerenciamento de Oficina Mecânica em C++

Sobre o projeto

Este projeto foi desenvolvido como trabalho acadêmico do curso de Análise e Desenvolvimento de Sistemas. O sistema simula o gerenciamento de uma oficina mecânica utilizando programação estruturada em C++.

A aplicação é executada em modo console e permite realizar o controle de clientes, veículos, mecânicos, peças, serviços e ordens de serviço, aplicando conceitos fundamentais de estruturas de dados e manipulação de registros em memória.

---

Funcionalidades

 - Cadastro e consulta de dados

* Visualização de cidades cadastradas;
* Visualização de serviços disponíveis;
* Consulta de peças em estoque;
* Cadastro de clientes com validação de código;
* Cadastro de veículos com validação de placa;
* Cadastro de mecânicos.

 - Controle de oficina

* Abertura de ordens de serviço;
* Associação de veículos aos seus clientes;
* Associação de mecânicos conforme especialidade;
* Registro de serviços realizados;
* Controle de utilização de peças e baixa automática no estoque.

 - Relatórios

* Consulta detalhada de peças;
* Relatório de peças abaixo do estoque mínimo;
* Cálculo do valor necessário para reposição do estoque;
* Cálculo do valor total arrecadado pelas ordens de serviço.

---

- Tecnologias utilizadas

* C++
* Programação estruturada
* Structs
* Vetores
* Funções
* Busca binária
* Manipulação de strings com `cstring`
* Entrada e saída de dados em console

---

 - Conceitos aplicados

Durante o desenvolvimento do projeto foram aplicados conceitos como:

* Organização de dados utilizando estruturas (`struct`);
* Modularização através de funções;
* Validação de dados de entrada;
* Pesquisa e ordenação de registros;
* Controle de estoque;
* Relacionamento entre entidades (cliente, veículo, serviço, peça e mecânico);
* Lógica de programação.

---

 - Como executar

1. Abra o arquivo `main.cpp` (ou o arquivo fonte do projeto) em uma IDE compatível com C++;
2. Compile o código utilizando um compilador C++;
3. Execute o programa pelo terminal ou pela própria IDE;
4. Utilize o menu interativo para navegar pelas funcionalidades do sistema.
